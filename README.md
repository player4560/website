# Green Palmeras – Front-End Demo
Prosta, statyczna strona internetowa prezentująca apartament wakacyjny w Mil Palmeras (Hiszpania).  
Projekt pokazuje praktyczne użycie **HTML5, CSS3 i JavaScript** oraz prosty system **wielojęzyczności (PL/EN/DE/ES)**.
Projekt ma charakter edukacyjny i demonstracyjny (podstawy front-endu).

## Funkcje
- Dynamiczna zmiana języka (JavaScript, `data-i18n`)
- Sekcje: apartament, galeria, okolica, kontakt
- Ikonografia i tabela wyposażenia
- Responsywny układ
- QR kody do social media

## Technologie
- **HTML5** – struktura strony  
- **CSS3** – stylizacja i layout  
- **JavaScript** – obsługa języków, manipulacja DOM  
- **Visual Studio** – środowisko pracy
